Параметры обучения ARIMAX модели
### Поля

| Поле | Схема | Описание                                             |
| ---- | ----- | ---------------------------------------------------- |
| P    | int   | порядок разностей (больше или равен нулю)            |
| D    | int   | порядок интегрирования (больше или равен нулю)       |
| Q    | int   | порядок скользящего среднего (больше или равен нулю) |
